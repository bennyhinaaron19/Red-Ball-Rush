A simple endless runner game built using React Native (Expo) where the player controls a blue rectange  and avoids falling obstacles.

## How to Run

1. Clone the repository:

git clone https://github.com/bennyhinaaron19/Red-Ball-Rush.git

2. Navigate to the project folder:

cd Red-Ball-Rush
cd SpaceEscapeRunner

3. Install dependencies:

npm install

4. Start the Expo development server:

npx expo start

5. Open the Expo Go app on your Android or iOS device and scan the QR code displayed in the terminal or Expo Developer Tools.
