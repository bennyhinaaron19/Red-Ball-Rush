A simple endless runner game built using React Native (Expo) where the player controls a blue rectange  and avoids falling obstacles.

How to Run 
Clone the repository:
git clone https://github.com/bennyhinaaron19/Red-Ball-Rush.git
Navigate to the project folder:
cd SpaceEscapeRunner
Install dependencies:
npm install
Start the Expo development server:
npx expo start
Open the Expo Go app on your mobile device and scan the QR code displayed in the terminal or Expo Developer Tools.
